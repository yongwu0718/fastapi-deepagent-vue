## Docker Desktop（Docker 桌面版）

Docker Desktop 是一款面向 Mac、Linux 或 Windows 环境的一键安装应用程序，可让您构建、共享和运行容器化应用程序及微服务。

它提供了一个简洁的 **GUI（图形用户界面）**，让您可以直接在本地机器上管理容器、应用程序和镜像。

Docker Desktop 减少了复杂设置所花费的时间，让您能够专注于编写代码。它会自动处理端口映射、文件系统相关事项以及其他默认设置，并定期通过错误修复和安全更新进行更新。

Docker Desktop 可与您偏好的开发工具和语言集成，并通过 **Docker Hub** 让您访问由可信镜像和模板组成的庞大生态系统。这使团队能够加速开发、自动化构建、启用 **CI/CD** 工作流，并通过共享仓库进行安全协作。

## 主要特性

* 能够在任何云平台上、使用多种语言和框架，对任意应用程序进行容器化并共享。
* 快速安装并设置完整的 Docker 开发环境。
* 包含最新版本的 **Kubernetes**。
* 在 Windows 上，能够在 Linux 容器与 Windows 容器之间切换，以便构建应用程序。
* 借助原生 Windows **Hyper-V** 虚拟化技术，实现快速可靠的性能。
* 在 Windows 机器上通过 **WSL 2** 原生支持 Linux 工作。
* 支持代码和数据的卷挂载，包括文件更改通知，以及通过 localhost 网络轻松访问运行中的容器。
