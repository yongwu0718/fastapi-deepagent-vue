教程 - 用户指南

本教程将一步步向你展示如何使用 FastAPI 的大部分功能。 

每一节内容都建立在之前的基础上，但其结构旨在将各个主题分离开来，以便你可以直接跳转到特定章节来解决你具体的 API 需求。 
它同时也旨在作为未来的参考资料，方便你随时回来查看所需的确切内容。

## 安装 FastAPI

第一步是安装 FastAPI。 请确保创建了一个  虚拟环境   ，激活它，然后安装 FastAPI。   

pip install "fastapi[standard]"
**注意** 

当你使用  pip install "fastapi[standard]"  安装时，它会附带一些默认的可选标准依赖项，包括  fastapi-cloud-cli ，它允许你部署到  FastAPI Cloud   。 

如果你不需要这些可选依赖项，则可以改为安装  pip install fastapi 。 

如果你想安装标准依赖项但不需要  fastapi-cloud-cli ，可以使用  pip install "fastapi[standard-no-fastapi-cloud-cli]"  进行安装。 

**提示** 

FastAPI 拥有一个  官方 VS Code 插件   （同时也支持 Cursor），它提供了许多功能，包括路径操作资源管理器、路径操作搜索、测试中的 CodeLens 导航（从测试跳转到定义），以及直接在编辑器中进行 FastAPI Cloud 部署和日志查看

## 进阶用户指南

在阅读完本教程 - 用户指南后，你还可以阅读进阶用户指南。

进阶用户指南建立在本书的基础上，使用了相同的概念，并教授你一些额外的功能。 

但你应该首先阅读教程 - 用户指南（即你现在正在阅读的内容）。 

它的设计初衷是让你仅通过教程 - 用户指南就能构建一个完整的应用程序，然后再根据你的需要，利用进阶用户指南中的一些额外想法以不同的方式对其进行扩展。
