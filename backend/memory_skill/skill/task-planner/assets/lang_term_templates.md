# 长期计划模板

输出长期计划时，使用以下格式：

```markdown
## 🗓 {title}

**时间范围**：{start_date} ~ {end_date}

### 目标与任务

| 目标 | 进度 | 截止 | 预警 | 任务 | 开始 | 结束 | 状态 |
|------|------|------|------|------|------|------|------|
| 📁 {goal_name} | {progress}% | {deadline} | {warning} |  ├ {task1} | {start} | {end} | 📌 |
|  |  |  |  |  ├ {task2} | {start} | {end} | 🔄 |
|  |  |  |  |  └ {task3} | {start} | {end} | ✅ |
| 📁 {goal_name2} | {progress}% | {deadline} | - |  ├ {task4} | {start} | {end} | 📌 |
```

## 预警规则

| 条件 | 预警文字 |
|------|---------|
| 已逾期 (deadline < today) | ⚠️ 已逾期 |
| 剩余 0-7 天 | ⏳ {n}天 |
| 剩余 8-30 天 | 📅 {n}天 |
| 剩余 > 30 天 | 不显示 |

## 状态图标

| 图标 | 状态 |
|------|------|
| 📌 | 计划中(planned) |
| 🔄 | 进行中(in-progress) |
| ✅ | 已完成(done) |

## 验证规则

| 检查项 | 通过标准 | 标记 |
|--------|---------|------|
| 目标全覆盖 | 步骤 4 所有输出任务均纳入目标下 | ✅/⚠ |
| 无孤立任务 | 每个任务归属明确的目标 | ✅/⚠ |
| 进度可量化 | 每个目标有百分比或里程碑进度 | ✅/⚠ |
| 预警生效 | 逾期和临近截止的任务已标记预警 | ✅/⚠ |
| 子任务时间窗合理 | 超过 3 个月的长期任务已拆分为月度子任务 | ✅/⚠ |
