# 日任务输出模板

输出每日计划时，使用以下格式：

```markdown
## 📅 {date} {day_of_week} · {theme}

### 🎯 核心目标
1. {goal_1} — 成果：{outcome_1}
2. {goal_2} — 成果：{outcome_2}

### 📋 时间块

| 时间 | 块 | 任务 | 精力 | 状态 | 时长 | 依赖 |
|------|----|------|------|------|------|------|
| 08:30-10:00 | 上午深度块 | 学习课程模块 3 | 🔵 | 📋 | 90min | - |
| 10:00-10:15 | ↳ 缓冲 | - | - | - | 15min | - |
| 10:15-11:00 | ↳ 深度延续 | 整理笔记 | 🔵 | 📋 | 45min | t1 |
| 11:00-11:30 | ↳ 收尾 | 回复邮件 | ⚪ | 📋 | 30min | - |
| 14:00-14:30 | 下午沟通块 | 团队周会 | 🟡 | 📋 | 30min | - |
| 14:30-15:30 | ↳ 深度 | 方案撰写 | 🔵 | 📋 | 60min | - |
| 15:45-16:30 | ↳ 沟通 | 接口对齐 | 🟡 | 📋 | 45min | t5 |
| 16:30-17:00 | 晚间整理块 | 收尾与明日计划 | ⚪ | 📋 | 30min | - |

### 📥 随手池
<!-- 微任务：<15分钟 + 单一动作 + 无依赖，不占周池容量，不参与重要性排序。每日清空。 -->
| 任务 | 时长 |
|------|------|
| 取快递 | 10min |
| 充话费 | 5min |

**总计划：360 分钟 | 缓冲：90 分钟 (20%)** ✅
```

## 精力图标

| 图标 | 类型 |
|------|------|
| 🔵 | 深度专注 |
| 🟡 | 沟通协作 |
| ⚪ | 机械执行 |
| 🟢 | 学习充电 |

## 状态图标

| 图标 | 状态 |
|------|------|
| 📋 | 计划中 |
| 🔄 | 进行中 |
| ✅ | 已完成 |
| ⏳ | 待定(backlog) |

## 验证规则

在输出表格后，逐项检查并标注：

| 检查项 | 通过标准 | 标记 |
|--------|---------|------|
| 任务完成定义明确 | 每个任务有可验证的产出 | ✅/⚠ |
| 无超大任务 | 无超过 2h 未分解的任务 | ✅/⚠ |
| 精力匹配 | 高精力任务匹配高效能时段 | ✅/⚠ |
| 缓冲充足 | 缓冲 >= 总时长的 20% | ✅/⚠ |
| 依赖合理 | 无循环依赖，前置任务已排期 | ✅/⚠ |
| 总量合理 | 每日总时长 <= 4.8h (288min) | ✅/⚠ |
| 深度连续 | 连续深度任务 <= 3.5h (210min) | ✅/⚠ |
