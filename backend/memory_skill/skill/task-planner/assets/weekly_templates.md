# 周计划模板

输出周计划时，使用以下格式：

```markdown
# 第 {周数} 周计划 ({起始日期} - {结束日期})

## 🎯 本周三大目标
<!-- 如果本周只能完成三件事，应该是这三件。它们通常与季度目标或个人长期目标关联。 -->
1. [目标 1]
2. [目标 2]
3. [目标 3]

## 📊 每日主题与重点
<!-- 为每天设定一个主题，将对应的高优先级任务分配进去。 -->
| 日期 | 主题 | 重点任务 | 预计时长 |
|------|------|---------|---------|
| 周一 | {主题} | {任务A} {任务B} | {n}h |
| 周二 | {主题} | {任务C} {任务D} | {n}h |
| 周三 | {主题} | {任务E} | {n}h |
| 周四 | {主题} | {任务F} {任务G} | {n}h |
| 周五 | {主题} | {任务H} | {n}h |
| 周末 | 弹性 | 缓冲或生活事务 | - |

## 🧩 跨日任务链
<!-- 列出需要多天连续推进的任务，标出依赖步骤和每日预计投入。 -->
| 任务链 | 最终产出 | 周一 | 周二 | 周三 | 周四 | 周五 |
|--------|---------|------|------|------|------|------|
| {链 1} | {产出} | 步骤 a (1h) | 步骤 b (2h) | - | 步骤 c (1h) | - |
| {链 2} | {产出} | - | 步骤 a (1.5h) | 步骤 b (2h) | - | 步骤 c (1h) |

## 📥 待分发任务池
<!-- 尚未确定具体日期、但需要在本周内完成的任务。 -->
| 任务 | 精力 | 预计时长 | 截止 | 前置条件 |
|------|------|---------|------|---------|
| 任务 X | 🔵 | 30min | - | - |
| 任务 Y | 🟡 | 1h | 周五 | 等任务 A 完成 |
| 任务 Z | 🔵 | 2h | 周四 | - |

## ⚖️ 一周负载概览
| 日期 | 总时长 | 🔵深度 | 🟡沟通 | ⚪杂务 | 🟢学习 | 缓冲% |
|------|--------|--------|--------|--------|--------|-------|
| 周一 | 4h | 2.5h | 1h | 0.5h | - | 20% |
| 周二 | ... | ... | ... | ... | ... | ... |
```

## 验证规则

| 检查项 | 通过标准 | 标记 |
|--------|---------|------|
| 日负载均衡 | 每日总时长相差不超过 1.5h | ✅/⚠ |
| 深度不连续超限 | 每日连续深度任务 <= 3h | ✅/⚠ |
| 跨日链完整 | 每条跨日链有明确的最终产出和每日步骤 | ✅/⚠ |
| 待分发池有上限 | 待分发池任务不超过 5 条 | ✅/⚠ |
| 三大目标关联 | 三大目标都有对应任务排入日程 | ✅/⚠ |
