# 数据约定

## 金额符号
- 支出金额为正值，收入金额为正值（工具返回的金额均为正数）

## 三类支出定义
- **刚性固定**：金额固定（房租、手机分期、保险）→ 关注异常波动和新增项
- **刚性必要**：必需但可变（押金、交通、医疗、话费、水电煤、中介费）→ 关注效率，期望控制在低位
- **弹性可选**：非必需（食品、饮品、购物、娱乐、服饰）→ 关注控制空间，是节流主战场

## 收入分类
- `真实收入` = 总收入 - 转账收款
- 分析净收支时应优先使用真实收入口径

## 数据完整性校验要点

在预测场景中，必须先做以下校验：

### 固定项校验
检查 `rigid_fixed_monthly` 最近 1-2 个月的明细项。若某项金额环比下降 > 40%，标记为 ⚠️ 疑似数据缺口/账期错位。

### 结构突变校验
对比 `monthly_categories` 中各月支出占比。若某月食品占比飙升至 45% 以上且刚性固定同步暴跌，说明总支出基数失真。

### 收入匹配校验
查看 `monthly_income_expense` 中的真实收入。若最近一月收入 < 历史月均的 70%，标记为 ⚠️ 收入断崖预警。
